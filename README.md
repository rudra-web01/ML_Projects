This repository contains beginner-friendly Machine Learning projects implemented using Python.
The goal of this repository is to practice fundamental ML concepts such as data preprocessing, visualization, model training, and evaluation.

Projects Included
1. House Price Prediction

A machine learning project that predicts house prices using Linear Regression.

Features

Data preprocessing

Train-test split

Linear Regression model

Model evaluation using R² score and error metrics

Technologies Used

Python

Pandas

NumPy

Scikit-learn

Matplotlib

Goal
To predict house prices based on input features using a supervised learning model.

2. Road Accident Analysis and Prediction

This project analyzes road accident data and predicts accident-related fatalities using Machine Learning.

Features

Data cleaning and preprocessing

Exploratory Data Analysis (EDA)

Accident data visualization

Scatter plots showing relationships between accident cases and deaths

Linear Regression model for prediction

Model evaluation using R² score, MSE, and RMSE

Visualizations Included

Accident cases vs deaths

Injuries vs deaths

Distribution of accident fatalities

Goal
To analyze patterns in accident data and predict fatalities based on accident statistics.

Technologies Used

Python

Pandas

NumPy

Matplotlib

Scikit-learn

Jupyter Notebook / Google Colab
